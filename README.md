# 🛡️ 보험콕검

**보험콕검**은 계약담당자가 제출받은 보험서류를 빠르게 확인할 수 있도록 돕는 **보험서류 검토 보조 웹앱**입니다.

건물 화재보험, 영업배상책임보험, 생산물배상책임보험 등 계약 과정에서 자주 확인해야 하는 보험서류를 업로드하고, 필요한 확인 항목을 한눈에 정리할 수 있도록 설계했습니다.

> 서류를 대신 판단하는 앱이 아니라,
> 담당자가 확인해야 할 지점을 놓치지 않도록 도와주는 검토 보조 도구입니다.

## 📌 만든 이유

계약 업무를 하다 보면 업체에서 제출한 보험서류가 실제로 필요한 조건을 충족하는지 확인해야 합니다.

예를 들어 다음과 같은 부분을 확인해야 합니다.

* 건물 화재보험 서류가 제출되었는지
* 영업배상책임보험 또는 생산물배상책임보험이 포함되어 있는지
* 보험기간이 계약기간과 맞는지
* 피보험자, 보험계약자, 사업장 주소가 맞는지
* 보상한도나 담보 항목이 서류에 표시되어 있는지

하지만 보험증권은 형식이 업체마다 다르고, 용어도 비슷비슷해서 처음 확인할 때 헷갈리기 쉽습니다.

**보험콕검**은 이런 확인 흐름을 정리해서
“무엇을 확인해야 하는지”와 “제출서류에 그 내용이 있는지”를 빠르게 점검할 수 있도록 만든 도구입니다.

## ✨ 주요 기능

### 보험서류 업로드

업체에서 제출한 보험 관련 PDF 파일을 업로드해 검토를 시작할 수 있습니다.

브라우저 안에서 파일을 읽고 분석하는 구조로, 사용자가 올린 서류가 별도 서버로 전송되지 않도록 설계했습니다.

### 필요서류 확인

계약담당자가 요구한 보험서류 기준으로 제출 여부를 정리할 수 있습니다.

예시 확인 항목은 다음과 같습니다.

* 건물 화재보험 가입증명서
* 영업배상책임보험 증명서
* 생산물배상책임보험 증명서
* 여행자보험 관련 확인 항목
* 기타 계약조건에 따른 보험서류

### 검토 결과 요약

제출된 서류에서 확인된 내용을 요약해 보여줍니다.

확인 결과는 단순히 “있음 / 없음”만 보여주는 것이 아니라,
담당자가 어떤 부분을 보고 판단해야 하는지 흐름을 따라갈 수 있도록 구성했습니다.

### 진행단계 표시

검토 과정에서 현재 어느 단계에 있는지 알 수 있도록 진행단계 영역을 제공합니다.

업로드 → 제출파일 확인 → 필요서류 확인 → 검토결과 요약 흐름으로
사용자가 화면에서 길을 잃지 않도록 하는 데 초점을 두었습니다.

### 브라우저 내부 분석

보험콕검은 민감한 보험서류를 다루는 도구이기 때문에
서버 저장이나 외부 전송 없이 브라우저 내부에서 분석하는 방향으로 설계했습니다.

## 🧾 확인할 수 있는 항목

보험콕검은 다음과 같은 항목을 확인하는 데 도움을 줍니다.

* 보험 종류
* 보험기간
* 보험계약자
* 피보험자
* 사업장 또는 목적물 주소
* 담보명
* 보상한도
* 증권번호
* 제출서류의 누락 여부
* 계약 조건과 관련된 확인 필요 사항

단, 보험 가입의 적정성이나 법적 판단은 담당자가 최종 확인해야 합니다.

---

## 💻 사용 방법

1. 보험콕검 웹앱에 접속합니다.
2. 검토할 보험서류 PDF를 업로드합니다.
3. 필요한 보험서류 항목을 확인합니다.
4. 제출된 파일에서 어떤 내용이 확인되었는지 검토합니다.
5. 최종적으로 계약담당자가 필요한 항목을 직접 확인합니다.

---

## 🗂️ 추천 사용 상황

보험콕검은 이런 상황에서 사용하기 좋습니다.

* 계약서류 검토 시 보험서류가 함께 제출된 경우
* 업체별 보험 가입증명서 제출 여부를 비교해야 하는 경우
* 건물 화재보험과 영업·생산물배상책임보험을 구분해 확인해야 하는 경우
* 보험증권에서 어떤 항목을 봐야 할지 정리하고 싶은 경우
* 제출서류 확인 결과를 간단히 정리해야 하는 경우

---

## ⚠️ 주의사항

보험콕검은 보험서류 검토를 돕는 보조 도구입니다.

다음 사항은 반드시 사용자가 직접 최종 확인해야 합니다.

* 계약 조건에 맞는 보험인지 여부
* 보상한도 충족 여부
* 보험기간과 계약기간의 일치 여부
* 피보험자 및 사업장 주소의 적정성
* 기관 내부 기준 또는 계약 특수조건 충족 여부

자동 분석 결과는 참고용이며, 최종 판단 책임은 사용자에게 있습니다.

## 🔒 개인정보 및 보안

보험콕검은 사용자가 업로드한 파일을 외부 서버로 전송하지 않는 것을 기본 방향으로 합니다.

* 서버 저장 없음
* 외부 전송 없음
* 브라우저 내부 분석
* 사용자가 직접 업로드한 파일만 검토

민감한 계약서류와 보험서류를 다루기 때문에
사용 전 기관 보안 기준에 맞게 활용해 주세요.

## 🚧 현재 버전

현재 버전은 초기 검토 보조 버전입니다.

보험서류 양식은 보험사와 업체마다 다를 수 있으므로,
일부 PDF에서는 텍스트 추출 결과가 정확하지 않을 수 있습니다.

향후 개선 예정 항목은 다음과 같습니다.

* 보험 종류별 확인 흐름 개선
* 제출파일 요약 영역 고도화
* 필요서류 확인 결과 표 개선
* 다양한 보험증권 양식 대응
* 사용자 안내 문구 개선
* 검토 결과 화면 가독성 개선
