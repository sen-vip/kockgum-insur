# 콕검 KOCKGUM

계약서류를 콕 집어 확인하는  
학교 행정실용 서류 검토 도구입니다.

계약 업무를 하다 보면  
서류는 여러 장인데, 확인해야 할 항목은 늘 정해져 있습니다.

콕검은 제출된 계약서류를 보면서  
필요한 서류가 있는지, 확인해야 할 항목이 빠지지 않았는지  
한눈에 체크할 수 있도록 만든 웹앱입니다.

## 🧾 서비스 소개

콕검은 계약서류 검토 과정에서  
담당자가 반복적으로 확인하는 항목을 정리해 주는 도구입니다.

계약명, 업체명, 보험증권, 여행자보험, 필수 확인 항목 등을  
체크리스트처럼 확인할 수 있도록 구성했습니다.

서류 내용을 대신 판단하거나 보장금액을 자동 판정하는 도구가 아니라,  
담당자가 확인해야 할 항목을 놓치지 않도록 도와주는 보조 도구입니다.

---

## 🧩 주요 기능

- 계약서류 확인 항목 정리
- 영업배상책임보험 증권 사본 확인
- 생산물배상책임보험 증권 사본 확인
- 여행자보험 제출 여부 확인
- 보험 관련 주요 확인 항목 안내
- 계약별 확인 결과 메모
- 학교마다 다른 기준은 담당자가 직접 확인할 수 있도록 보조
- 브라우저에서 바로 실행되는 정적 웹앱

---

## 🛡️ 확인 범위 안내

콕검은 서류의 존재 여부와 확인해야 할 항목을 정리하는 도구입니다.

특히 보험 관련 내용은 학교, 계약 유형, 사업 내용에 따라  
필요 기준이나 보상한도가 달라질 수 있습니다.

따라서 콕검은 다음과 같은 방식으로 사용합니다.

- 서류가 제출되었는지 확인
- 확인해야 할 항목이 문서에 있는지 확인
- 보상한도, 가입금액, 유효기간 등은 담당자가 직접 검토
- 최종 적정성 판단은 계약담당자가 수행

콕검은 계약 검토를 자동으로 확정하는 도구가 아니라,  
확인 누락을 줄이기 위한 보조 도구입니다.

---

## 🖥️ 사용 방법

1. 콕검 페이지에 접속합니다.
2. 계약서류 확인 항목을 순서대로 살펴봅니다.
3. 제출된 서류에 해당 내용이 있는지 확인합니다.
4. 필요한 경우 확인 메모를 남깁니다.
5. 최종 판단은 계약 기준과 학교 내부 방침에 따라 담당자가 확인합니다.

## 🔒 저장 및 보안 안내

콕검은 GitHub Pages에서 실행되는 정적 웹앱입니다.

기본적으로 별도 서버 저장 기능이나 로그인 기능이 없습니다.  
입력한 내용은 외부 서버로 자동 전송되지 않으며,  
브라우저 화면에서 확인용으로 사용하는 것을 전제로 합니다.

다만 실제 계약서류에는 개인정보나 민감한 정보가 포함될 수 있으므로,  
공용 PC 사용 시 입력 내용과 화면 노출에 주의해 주세요.

## 🌱 만든 이유

계약서류 검토는 단순해 보이지만  
실제로는 놓치면 안 되는 항목이 많습니다.

특히 보험증권, 여행자보험, 유효기간, 보장 항목처럼  
반복해서 확인해야 하는 내용은 담당자에게 은근히 부담이 됩니다.

콕검은 그런 반복 확인 업무를 조금 더 가볍게 만들기 위해 만들었습니다.

“이 서류 있었나?”  
“이 항목 확인했나?”  
“학교 기준은 따로 봐야 하는데, 일단 내용은 있나?”

이런 순간에 빠르게 열어두고 쓰는  
작은 계약서류 확인 도구를 목표로 합니다.

## ⚠️ 이용 시 주의사항

콕검은 행정업무 보조용 도구입니다.

법적 검토, 계약 적정성 판단, 보험 가입 적정성 판단을 대신하지 않습니다.  
최종 확인과 책임은 해당 업무 담당자 및 기관 기준에 따릅니다.

실제 업무에 적용할 때는  
학교 내부 지침, 계약 조건, 관련 법령, 교육청 안내 사항을 함께 확인해 주세요.
